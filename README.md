## 👨‍💻 How to Setup

```bash
  # Clone the project
  $ git clone https://github.com/Ezever/Projeto-NASA.git
```
```bash
  # Enter directory
  $ cd Projeto-NASA
```

```bash
  # Install the dependencies, if you use npm
  $ npm install
```
<p align="center">or</p>

```bash
  # Install the dependencies, if you use yarn
  $ yarn
```
